== Informationen zur Priorität:

Wenn du die Priorität änderst, entscheidest du, wann ChatEx in den Chat eingreift.
Der standard Wert LOWEST wird als erstes ausgeführt.
Es gibt fünf verschiedene Prioritäten, die man benutzen kann.
LOWEST, LOW, NORMAL, HIGH und HIGHEST.
LOWEST wird dabei als erstes ausgeführt und HIGHEST als letztes.
Wenn du nun Multiverse-Core installiert hast und möchtest, dass es deinen Chat mit dem Name der Welt prefixt, solltest du es auf LOW oder
LOWEST setzen. Denn:
Multiverse nimmt das Format was bereits gesetzt wurde und setzt den Weltnamen davor. Danach wird das Format wieder gesetzt.
Wenn jetzt aber ChatEx nach Multiverse-Core eingreift, wird das Prefix von Multiverse-Core mit dem von ChatEx überschrieben.



== Multi-prefixes und -suffixes

!!!This feature funktioniert im Moment nur mit PermissionsEx!!!

Wenn du das Multi-prefixes/suffixes feature aktiviert hast, bekommt der Spieler das Prefix von allen Gruppen in denen er ist.
Beispiel:

Du hast zweui Gruppen:
Owner und Moderator.

Das Prefix von Owner ist '&4Owner ' und das Prefix von Moderator ist '&9Moderator '.
Du bist in beiden Gruppen.
Dann wäre dein Prefix: '&4Owner &9Moderator '.
